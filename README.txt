Compatibility: UI Widget Builder supports Unreal Engine UE 5.6 / 5.7 / 5.8. Enable the Unreal CommonUI plugin before using Common Animated Switcher imports in UE 5.7 / UE 5.8. The importer can save the preset path for faster repeated preset loading/saving.
